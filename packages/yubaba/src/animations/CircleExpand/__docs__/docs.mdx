# CircleExpand

Will animate a circle from the origin element to cover the entire viewport,
and then fade out.

Generally you should use [CircleExpand](/circle-expand) and [CricleShrink](/circle-shrink) together to seamlessly transition the background between pages.

## Usage

```js
import { CircleExpand } from 'yubaba';
```

<Playground>
  <Common.Toggler>
    {({ shown, toggle }) =>
      !shown ? (
        <Styled.Container>
          <Baba name="circle-expand" key="1">
            <CircleExpand background="#ff5e6d">
              {({ ref, style }) => (
                <Styled.Button onClick={() => toggle()} style={style} innerRef={ref} />
              )}
            </CircleExpand>
          </Baba>
        </Styled.Container>
      ) : (
        <Styled.Container onClick={() => toggle()} interactive>
          <Baba name="circle-expand" key="2">
            {({ ref, style }) => <div style={style} ref={ref} />}
          </Baba>
        </Styled.Container>
      )
    }
  </Common.Toggler>
</Playground>
